PROBLEM
- Build a program that gets length and width of a room in meters. 
- have the program display the area of the room in both square feet and meters.
- Input: float 
- Output: string containing result as float

EXAMPLE
Enter the length of the room in meters:
10
Enter the width of the room in meters:
7
The area of the room is 70.0 square meters (753.47 square feet).

DATA STRUCTURE
-input: float
-output: float

ALGORITHM
1. Create a message to receive input of length in meters.
2. create a message to recive input of width in meters. 
3. create a variable named answer_in_meters and have the sum of length * width saved to this.
4. create a variable name anser_in_feet which is the result of answer_in_meters * 10.7639.
5. print out the answer message containing answer_in_meters and answer_in_feet.
